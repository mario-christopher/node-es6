# node-es6
**Node ES6** is a simple code sample that shows how to use all of ES6 features in a Node application.

## Setup
* `git clone https://gitlab.com/mario-christopher/node-es6.git`
* `cd node-es6`
* `npm install`
* `npm run server`

##  Application features:

* Use of ES6 features.
